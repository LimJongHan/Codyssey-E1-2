# Codyssey E1-2 · Git과 함께하는 Python 첫 발자국

터미널에서 동작하는 **나만의 퀴즈 게임**입니다. Python 기본 문법과 클래스(객체 지향)로 역할을 나누고, `state.json`(UTF-8)에 저장해 **프로그램을 종료해도 퀴즈와 최고 점수가 유지**되도록 만들었습니다. Git으로 기능 단위 커밋·브랜치 생성·병합을 연습하는 것을 목표로 합니다.

## 프로젝트 개요

- **언어/버전**: Python 3.10+
- **실행 형태**: 콘솔 프로그램
- **데이터 저장**: 프로젝트 루트의 `state.json` (UTF-8)
- **외부 라이브러리**: 사용하지 않음(표준 라이브러리만)

## 퀴즈 주제 선정 이유

- **주제**: 기초 산수(덧셈/뺄셈/곱셈/나눗셈)
- **이유**: 정답이 명확해서 로직 검증이 쉽고, 4지선다 + 정답 번호(1~4) 규칙을 일관되게 적용하기 좋습니다.
- **기본 포함 문제 수**: 5문제 이상

## 실행 방법

프로젝트 루트에서:

```bash
python main.py
```

## 기능 목록

- **퀴즈 풀기(메뉴 1)**: 저장된 퀴즈를 순서대로 출제하고 정답/오답을 표시한 뒤 결과를 출력합니다.
- **퀴즈 추가(메뉴 2)**: 문제/선택지 4개/정답(1~4)을 입력받아 저장합니다.
- **퀴즈 목록(메뉴 3)**: 등록된 퀴즈의 “문제(질문)”만 목록으로 확인합니다.
- **점수 확인(메뉴 4)**: 최고 점수(`best_score`)를 확인합니다. 기록이 없으면 안내합니다.
- **종료(메뉴 5)**: 프로그램을 종료합니다.

## 데이터 파일 설명 (`state.json`)

- **경로**: 프로젝트 루트의 `state.json`
- **인코딩**: UTF-8
- **역할**: 퀴즈 목록(`quizzes`)과 최고 점수(`best_score`)를 저장하여 재실행 시에도 유지합니다.

### 스키마(필드)

```json
{
  "version": 1,
  "quizzes": [
    { "question": "문제", "choices": ["보기1", "보기2", "보기3", "보기4"], "answer": 1 }
  ],
  "best_score": 100,
  "best_correct": 5,
  "best_total": 5
}
```

- **`quizzes`**: 퀴즈 목록
- **`best_score`**: 최고 점수(백분율, 0~100). 아직 기록이 없으면 `null`
- **`best_correct` / `best_total`**: 최고 점수 당시 정답 수/총 문항 수

### 예외 처리

- `state.json`이 **없거나 비어 있으면** 기본 데이터로 생성합니다.
- `state.json`이 **손상된 JSON이면** 안내 메시지 후 기본 데이터로 복구합니다.

## Git 체크리스트(제출용)

- **의미 있는 커밋 10개 이상**: 기능 단위로 커밋 + 브랜치/머지 로그 포함
- **브랜치 생성/병합 기록**: `git log --oneline --graph --decorate` 캡처
- **clone/pull 실습**:
  - 다른 폴더에 `git clone <repo>`로 복제
  - 복제본에서 README 한 줄 수정 → commit → push
  - 원래 폴더에서 `git pull`로 변경 반영
